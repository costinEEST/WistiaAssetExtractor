Help & Usage Guide

How to use this tool:

  1. Enter a Wistia video ID, full URL, or embed code in the input field
  2. Click "Extract Assets" to fetch available video assets
  3. Use the table to view, copy, or download individual assets
  4. Use bulk actions to select multiple assets for download
  5. Export asset information as CSV or JSON

Supported input formats:

  • Video ID: abc123def456
  • Full URL: https://example.wistia.com/medias/abc123def456
  • Embed code: Extract from iframe src or wvideo parameter
